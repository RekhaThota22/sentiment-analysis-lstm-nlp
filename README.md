# Sentiment Analysis using LSTM

This project builds an end-to-end sentiment analysis model using deep learning.

## Problem
Classify text reviews into Positive or Negative sentiment.

## Approach
- Text preprocessing and cleaning
- Tokenization and sequence padding
- Model building using RNN and LSTM
- Hyperparameter tuning
- Prediction pipeline for new inputs

## Tech Stack
- Python
- TensorFlow / Keras
- NLP

## Results
The model predicts sentiment of input text.

### Example:
Input: "This product is amazing"  
Output: Positive  

Input: "Worst product ever"  
Output: Negative
